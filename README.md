# Автотест футера Only Digital
Этот проект содержит простой автотест на Selenium, который проверяет:
- наличие футера
- наличие логотипа в футере
- наличие ссылки "Контакты"
- наличие любых навигационных ссылок

## Запуск
1. Установить зависимости:
   pip install -r requirements.txt
2. Запуск теста:
   python test_footer.py
